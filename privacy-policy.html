Privacy Policy for RemindSafe Last updated: March 2026 RemindSafe respects your privacy and is designed to keep your personal information secure. This Privacy Policy explains how the application handles user data. 1. Information Collection RemindSafe does not collect, store, or transmit personal information to external servers. The application primarily works offline and keeps all user data on the user's device. 2. Local Data Storage All reminders, notes, and documents stored in the RemindSafe vault remain locally on the user’s device. The application does not upload or store this information on remote servers. 3. Permissions Used RemindSafe may request certain device permissions strictly to provide core functionality: Storage / File Access This permission allows the application to save documents, create backup files, and restore backups selected by the user. Notifications This permission allows the application to send reminder notifications for events and tasks created by the user. These permissions are used only for the features described above and not for collecting personal data. 4. Data Sharing RemindSafe does not automatically share user data with any third parties. Users may manually share reminders or content using external applications if they choose. 5. Backup and Restore Users can export or import backups of their data. Backup files are stored only in locations selected by the user on their device or storage service. 6. Security RemindSafe is designed to keep user data secure on the device. Any security features such as device authentication (PIN, password, or biometric lock) depend on the user’s device settings. 7. Children’s Privacy RemindSafe does not knowingly collect personal information from children. Since the application does not collect user data, it does not store information about users of any age. 8. Changes to This Privacy Policy This Privacy Policy may be updated from time to time to reflect improvements or changes in the application. Updates will be reflected by revising the "Last updated" date. 9. Contact If you have any questions or concerns about this Privacy Policy, you can contact: Email: remindsafe.app@gmail.com
